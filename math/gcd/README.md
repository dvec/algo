# НОД (GCD)
НОД(`a`, `b`) - это наибольшее число, которое одновременно делит и `a`, и `b`
### Алгоритм
Алгоритм настолько прост, что его можно понять из кода, который можно найти ниже
### Реализация
#### Асимптотика: O(log min(a, b))
##### Входные данные:
 - a - первое число
 - b - второе число
##### Выходные данные:
 - r - НОД `a` и `b`
```python3
def gcd(a, b):
    while (b):
        a %= b
        a, b = b, a
    return a
```
